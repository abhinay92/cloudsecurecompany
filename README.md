# cloudsecurecompany

Requirements
------------

1. Design a cloud landing zone using Account Factory
2. Spoke Accounts should be hardened as per CIS standards
3. A centralized Security Account should be present where all the security hub, guardduty should be aggregated
4. Deploy kubernetes clusters in one of the spoke account using eksctl
5. Harden the cluster to meet CIS standards
6. Deploy a bunch of automations for secret management, log events processing and centralised to SIEM
